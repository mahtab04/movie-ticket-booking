# movie-ticket-booking

## Book your movie ticket here

### Description

    This is a simple movie ticket booking application.This is desktop based GUI application.

    You can book tickets, cancel tickets
### Tech Stack
- C++
- Qt
- Sqlite
- Qt Creator
- Qt Designer
- Qt Style Sheet

### Features
- Book tickets
- Cancel tickets
- Delete tickets
